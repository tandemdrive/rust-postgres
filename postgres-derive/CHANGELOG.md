# Change Log

## v0.4.6 - 2024-09-15

### Changed

* Upgraded `heck`.

## v0.4.5 - 2023-08-19

### Added

* Added a `rename_all` option for enum and struct derives.
* Added an `allow_mismatch` option to disable strict enum variant checks against the Postgres type.

### Changed

## v0.4.2 - 2022-04-30

### Added
